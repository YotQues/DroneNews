# DroneNews

 # Run Backend
 From the solution root directory
 ```sh
$ dotnet run --project="./DroneNews.Server/DroneNews.Server.csproj"
```
 
 # Run Client
 From the solution root directory
 ```sh
$ cd ./dronenews.client
$ npm run dev
```

# Test Client
From the solution root directory
 ```sh
$ cd ./dronenews.client
$ npm run test
```
